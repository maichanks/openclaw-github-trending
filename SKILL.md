# GitHub Trending Notifier

每小时自动检查 GitHub Trending，生成 AI 摘要并推送到指定频道。

## Features

- 🔍 获取 daily/weekly Trending Repos
- 🤖 AI 生成亮点摘要（使用 OpenRouter）
- 📢 推送到 Telegram / Feishu / Discord
- ⏰ 支持定时任务（cron）

## Quick Start

```bash
# 1. 配置环境变量
export GITHUB_TOKEN="ghp_xxx"
export OPENROUTER_API_KEY="sk-xxx"

# 2. 运行一次
node scripts/trending.js

# 3. 设置定时（OpenClaw cron）
# 每小时运行
```

## Output

精美卡片消息，包含：
- Repo 名称和描述
- 星数、语言、fork 数
- AI 总结的亮点
- 直达链接

## Configuration

| 环境变量 | 说明 | 必填 |
|---------|------|------|
| `GITHUB_TOKEN` | GitHub PAT（repo 权限） | ✅ |
| `OPENROUTER_API_KEY` | OpenRouter API Key | ✅ |
| `NOTIFY_CHANNEL` | 通知渠道（telegram/feishu/discord） | ✅ |
| `NOTIFY_TARGET` | 目标 chat/user ID | ✅ |
